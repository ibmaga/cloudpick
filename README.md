# cloudpick

Перебирает публичные IP на **cloud.ru Evolution** до попадания в нужную подсеть.

- Поддержка всех трёх зон (ru.AZ-1, ru.AZ-2, ru.AZ-3) — перебирает по очереди
- Интерактивный ввод параметров при запуске
- Сохраняет интересные IP (46.x) и не удаляет их
- Retry на удаление при таймаутах
- Работает из любого места на сервере после установки

## Установка

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/ibmaga/cloudpick/main/install.sh)
```

## Запуск

```bash
cloudpick
```

Или с переменными окружения (чтобы не вводить каждый раз):

```bash
export CLOUDRU_KEY_ID=...
export CLOUDRU_KEY_SECRET=...
export CLOUDRU_PROJECT_ID=...
cloudpick
```

## Параметры при запуске

Скрипт спросит:

| Параметр | Описание |
|---|---|
| Key ID / Key Secret | Сервисный аккаунт cloud.ru (API → Сервисные аккаунты) |
| Project ID | ID проекта (виден в URL консоли) |
| Целевая подсеть | Например `46.243.142.0/23` |
| AZ ID для каждой зоны | Перехватить из DevTools при аренде IP (можно пропустить) |
| Паузы и кол-во попыток | Настройка скорости перебора |

## Как найти AZ ID

1. Открой консоль cloud.ru → Публичные IP
2. DevTools (F12) → Network → XHR
3. Нажми «Арендовать публичный IP», выбери нужную зону
4. Найди POST-запрос на `/floating-ips` → скопируй `availability_zone_id` из тела

Известные AZ ID:
- `ru.AZ-2` = `479a4ab3-3ff3-4972-95c5-7610bac5c0bb`
